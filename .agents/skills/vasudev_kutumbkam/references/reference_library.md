## 📌 Guidelines for Adding Links

1. **Verify the link is live** before adding
2. **Primary sources preferred** — official org sites, peer-reviewed papers, government data
3. **Write a clear description** — tell the visitor what they'll find in 1-2 sentences
4. **Group correctly** — if unsure, default to "Inspiration & Perspectives"
5. **Update this file** when adding to `index.html`
6. **Commit and push** after every change so the live site stays current
